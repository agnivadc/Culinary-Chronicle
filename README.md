## üõ°Ô∏è Security Protocol (Crucial)

To enable the AI intelligence without exposing your credentials, you **must** configure your API key in your host's dashboard:

1. **Obtain Key**: Get your API key from [Google AI Studio](https://aistudio.google.com/).
2. **Configure Host**:
   - **Vercel**: Project Settings > Environment Variables > Add `API_KEY`.
   - **Netlify**: Site Configuration > Environment Variables > Add `API_KEY`.
   - **GitHub**: Settings > Secrets & Variables > Actions > Add `API_KEY`.
